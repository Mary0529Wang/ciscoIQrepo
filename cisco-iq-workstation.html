Cisco IQ 是什么？

Cisco IQ 是面向客户的 AI 驱动数字化体验平台，连接客户与 Cisco Services 的完整旅程。它将实时洞察、评估、故障排除、学习、自动化和智能助手整合在统一界面中，帮助客户规划、部署、管理、保护和优化其 IT 环境。

Cisco IQ 现已通过 SaaS 提供，On-Prem（Tethered / Air-Gapped）模式即将推出。持有有效 Cisco Support 合同的客户可免费使用，无额外费用。

核心概念

应用模块、支持层级与部署模式速查。

四大核心应用

Assets

动态资产清单，AI 洞察风险趋势。包含 Inventory、EOL、Service Contracts、LDOS Dashboard 等。

Assessments

智能安全公告、设备加固（Security Hardening）、配置最佳实践评估与可操作建议。

Support

简化案例开立、案例摘要、智能案例管理，与 TAC 流程深度集成。

AI Assistant

对话式界面，支持查询、分析、信息综合，以及自助故障排除与报告生成。

支持层级（Support Tiers）

能力逐级累加：Signature 包含 Standard 全部能力，Standard 包含 Basic 全部能力。部分高级 Insights 需通过 Cisco IQ Link、Intersight、Meraki、SD-WAN Manager 或 Webex Control Hub 连接遥测数据。

Basic

Know what you have
  • Asset Inventory / EOL Reports
  • Service Coverage Reports
  • LDOS Dashboard
  • Security Advisories / Field Notices
  • Case Management
  • Self-serve Troubleshooting

Standard

Prioritize for operational resilience
  • EOL / LDOS / Coverage Insights
  • Asset Criticality Insights
  • Asset Tagging
  • Security Advisory Insights
  • Security Hardening Insights
  • Configuration Insights

Signature

Accelerate operational excellence
  • Configuration Recommendations
  • Security Hardening Recommendations
  • 专属专家团队与 SLA 保障
  • 主动安全加固与根因消除
部署模式

SaaS

云优先策略，快速部署。建议部署 On-Prem 的 Cisco IQ Link 作为安全桥接，获取最优 AI 洞察。

On-Prem Tethered

数据驻留本地，通过 tethered 连接获取云端更新与新功能，兼顾安全与便利。

On-Prem Air-Gapped

完全离线，适用于高合规行业。所有更新需手动操作，无外部网络连接。

Cisco IQ Link

Cisco IQ Link 是部署在客户 On-Prem 网络的虚拟机组件，负责自动设备发现与遥测采集，并可连接 Catalyst Center。Cisco IQ Connector 是 IQ Link 内部的采集组件，二者集成但职责不同：Link 提供 SaaS 与 On-Prem 之间的安全通信通道。

采集器非强制： 可仅用合同数据建立初始清单；高级分析与部署数据需安装 IQ Link 或直接产品遥测。

管理员入门三步

1. 连接云产品

连接 Intersight、Meraki、SD-WAN、Webex 等，最快获得个性化洞察。

2. 添加服务合同

整合各团队成员合同，覆盖未通过遥测连接的资产。

3. 配置 IQ Link

为 On-Prem 设备建立连接，解锁完整 AI 洞察能力。
