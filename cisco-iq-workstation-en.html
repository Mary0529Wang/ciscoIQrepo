Quick Start
What is Cisco IQ?

Cisco IQ is the customer-facing, AI-powered digital experience that connects the full customer journey with Cisco Services. It unifies real-time insights, assessments, troubleshooting, learning, automation, and the AI Assistant in a single interface — helping customers plan, deploy, manage, secure, and optimize their IT environments.

Cisco IQ is available via SaaS today; On-Prem (Tethered / Air-Gapped) modes are coming soon. Customers with a valid Cisco Support contract can use it at no additional cost.

Core Concepts

Quick reference for applications, support tiers, and deployment modes.

Four Core Applications

Assets

Dynamic asset inventory with AI-driven risk insights. Includes Inventory, EOL, Service Contracts, LDOS Dashboard, and more.

Assessments

Intelligent security advisories, device hardening, configuration best-practice assessments, and actionable recommendations.

Support

Simplified case creation, case summarization, and intelligent case management — deeply integrated with TAC workflows.

AI Assistant

Conversational interface for queries, analysis, synthesis, self-serve troubleshooting, and report generation.

Support Tiers

Capabilities are cumulative: Signature includes all Standard features; Standard includes all Basic features. Advanced Insights require telemetry via Cisco IQ Link, Intersight, Meraki, SD-WAN Manager, or Webex Control Hub.

Deployment Modes

SaaS

Cloud-first, rapid deployment. Deploy on-prem Cisco IQ Link as a secure bridge for optimal AI insights.

On-Prem Tethered

Data stays on-premises; tethered connection delivers cloud updates and new features — security plus convenience.

On-Prem Air-Gapped

Fully offline for highly regulated industries. All updates are manual; no external network connectivity.

Cisco IQ Link

Cisco IQ Link is a VM component deployed in the customer's on-premises network for automated device discovery and telemetry collection. It can connect to Catalyst Center. Cisco IQ Connector is the collection component inside IQ Link — integrated but distinct: Link provides the secure channel between SaaS and on-prem environments.

Collector is optional: Initial inventory can be built from contract data alone; advanced analytics require IQ Link or direct product telemetry.

Admin Onboarding — Three Steps

1. Connect Cloud Products

Link Intersight, Meraki, SD-WAN, Webex, etc. for personalized insights in minutes.

2. Add Service Contracts

Consolidate team contracts and cover assets not connected via telemetry.

3. Configure IQ Link

Connect on-prem devices and unlock full AI insight capabilities.
